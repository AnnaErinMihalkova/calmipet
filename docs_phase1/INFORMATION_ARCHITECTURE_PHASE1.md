# Information Architecture — Phase 1

- App
  - Home (Dashboard)
    - Header
      - Theme toggle
      - Welcome text
      - Add Test Reading button
    - Cards
      - Pet card (mood, streak, accessory)
      - Trend card (last 20 BPM bars)
      - Coach card (start breathing)
      - Recent Readings list
    - Overlay
      - Breathing coach (timer + cycle)
  - Auth
    - Sign Up
      - Username, email, password, confirm
      - Show/hide password toggles
    - Log In
      - Email, password
      - Show/hide password toggle
  - Onboarding
    - Info form
      - Age, gender, baseline HR
      - Stress rating (1–10)

- Navigation
  - Landing → Sign Up / Log In → Onboarding → Home
